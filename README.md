In-depth documentation on swappi is available at [docs.swappi.finance](https://docs.swappi.finance/).

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
